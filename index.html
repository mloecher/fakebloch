A work in progress MRI simulation website, where the Bloch equations and rendering is all done client side with javascript (typescript).

The main goal of this project is to have an educational site where people can get a better idea of how MRI works, without having to install software and with a (hopefully) usable UI.

Written by: Mike Loecher
Goals:
  • The goal here is to have a bunch of preset settings for various lessons
Goals:

Goals:
